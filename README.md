## Setup

```
git clone https://github.com/EvaBro/captain_cortex_paradigm
```

### Stimuli
Please contact the author for access to the stimuli.

### Dependencies
This paradigm requires [stim_utils](https://github.com/EvaBro/stim_utils). Clone it as a sibling folder:

```
git clone https://github.com/EvaBro/stim_utils
```

### Final folder structure
```
YourFolder/
├── stim_utils/
└── captain_cortex_paradigm/
    └── circles_enlarged
    └── nontarget_images
    └── captain_image
    └── instructions
```
