## Paradigm
Trials consist of still and moving circles to elicit visual gamma oscillations. After presentation of the circles, an image is shown. Participants are asked to press a button if the image is that of an astronaut (Captain Cortex), to maintain attention. 

## Setup

```
git clone https://github.com/EvaBro/captain_cortex_paradigm
```

### Stimuli
Please contact the author for access to the stimuli.

### Dependencies
This paradigm requires [stim_utils](https://github.com/EvaBro/stim_utils). Clone it as a sibling folder:

```
git clone https://github.com/EvaBro/stim_utils
```
